CSS

CSS is Cascading Style Sheets, which is used to design a HTML/web page like layout, design of page and also in display for different screen size we can make changes.

CSS Syntax:

Selector { Declaration; Declaration;}

About tags:

Tag Purpose
border To specify table borders in CSS
width Span the entire screen (full-width)
border-collapse The table borders collapsed into a single border
text-align Sets the horizontal alignment (like left, right, or center) of the content
tr Tag defines a row in an HTML table
td tag defines a standard data cell in an HTML table
th tag defines a header cell in an HTML table
